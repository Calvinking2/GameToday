## About the Game Project 

Game Made for Game Today competition, Richard As Programmer, David as 2D artist, Calvin as Game Design/Map artist
You are playing 3 robot that are being tested, complete the level at all cost
even if it mean sacrifice each other

#### Character
- Square Bot
  Have the ability to punch, making  everything in it's way fly upward
- Circle Bot
  Have the ability to fire bubble, this bubble is hard enough to stop a bullet making it a good platform to step on
- Triangle Bot
  Have the ability to shoot laser, making anything hit fly away far

#### Game controls

The following controls are bound in-game, for gameplay and testing.

note : Key Binding are subject to change

| Key Binding       | Function          |
| ----------------- | ----------------- |
| A,D           | Standard movement |
| Space             | Jump              |
| Left Click        | Attack             |
